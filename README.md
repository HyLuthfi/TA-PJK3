## Deskripsi Praktikum

Praktikum ini membahas tentang konfigurasi VLAN (Virtual Local Area Network) dan trunking menggunakan Cisco Packet Tracer.

## Topologi

### Perangkat
* Switch: 2 unit (S1, S2)
* PC: 3 unit (PC-A, PC-B, PC-C)

### VLAN yang Dikonfigurasi
* **VLAN 10:** Operations
* **VLAN 20:** Parking_Lot
* **VLAN 99:** Management
* **VLAN 1000:** Native

## Tujuan Praktikum

1. Membangun topologi jaringan dan konfigurasi dasar
2. Membuat VLAN dan assign port switch
3. Melakukan maintenance VLAN
4. Mengkonfigurasi 802.1Q trunk antar switch
